# VR-Boat
## Description 
A Virtual Reality game/scenario on a Boat, exploring the melting icebergs and polluted oceans. 

Built during VR Hackathon 20-21/08/2022.

### Topics:
- VR Virtual-Reality 
- Climate Change

## Result
![Result](./img/result.jpg)

## Requirements & Dependencies
- Unity 2021.3
- Oculus Quest 2

## Install 
*   clone the repository
*   open with UnityHub
*   connect your device (in developer mode) with a usb cable
*   build and run the app

## Run
Download the .apk in [releases](https://github.com/igor-lirussi/VR-Boat/releases) and install it in the VR Headset
